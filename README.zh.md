# DriverIndexer

## 介绍

`DriverIndexer` 是用于创建、读取和安装驱动包索引的工具。

## 软件架构

使用`Rust`开发，调用`devcon.exe`获取硬件id、安装驱动。

## 使用说明

1. 打开cmd
2. 输入本程序路径即可查看使用说明
